# Read It Out Privacy Policy

Last updated: May 28, 2026

Read It Out reads selected text and page text aloud in your browser. The extension does not run ads, sell user data, or use analytics.

## Data Stored Locally

Read It Out stores settings in browser extension storage, including:

- Selected speech engine
- Playback speed
- Page control preferences
- Voice, model, and language preferences
- API keys you enter for ElevenLabs, OpenAI, Sarvam AI, or Smallest AI

These settings remain in your browser unless you remove the extension data.

## Text Sent To Speech Providers

When you use Browser Speech, text is handled by your browser's built-in speech engine.

When you use Kokoro Local in the Chrome build, speech is generated with bundled local model files in the extension.

When you choose ElevenLabs, OpenAI, Sarvam AI, or Smallest AI, Read It Out sends the text you ask it to read, plus the API key and selected voice/model settings needed for the request, to that selected provider. Read It Out does not send text to other providers.

Provider processing is governed by the provider's own terms and privacy policy.

## Website Access

Read It Out runs on web pages so it can show the selection mini player, extract readable page text, and highlight the current text while reading. Page text is processed locally unless you choose a cloud speech provider for playback.

## Sharing And Sale

Read It Out does not sell, rent, or share user data for advertising or profiling. Data is used only to provide the speech playback features requested by the user.

## Security

API requests use HTTPS. API keys are stored in browser extension storage and are sent only to the speech provider selected in settings.

## Chrome Web Store Limited Use

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## Contact

For privacy questions, open an issue in the project repository.
